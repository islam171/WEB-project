# CLONE
git clone https://github.com/ваш-username/movie-app.git

# LAUNCHING ANGULAR
1) ```cd movie-app-frontend```
2) ```npm install```
3) ```ng serve```

# LAUNCHING DJANGO
1) ```cd movie-app-backend```
2) ```For Windows: > venv\Scripts\activate```
```For Mac: source venv/bin/activate```
3) ```python manage.py migrate```
4) ```python manage.py runserver```
